MORSE-500: A Programmatically Controllable Video Benchmark to Stress-Test Multimodal Reasoning

Zikui Cai, Andrew Wang, Anirudh Satheesh, Ankit Nakhawa, Hyunwoo Jae, Keenan Powell, Minghui Liu, Neel Jay, Sungbin Oh, Xiyao Wang, Yongyuan Liang, Tom Goldstein, Furong Huang
University of Maryland, College Park
Paper arXiv Code Data

MORSE-500 provides 500 programmatically generated video instances across six reasoning categories: mathematical, abstract, spatial, temporal, physical, and planning.

Example Videos

Key Features

MORSE-500 addresses critical limitations in existing multimodal reasoning benchmarks through several key innovations:

  • Temporal Complexity: Video-based tasks that require understanding dynamic sequences and causal relationships over time
  • Diverse Reasoning: Six complementary categories spanning mathematical, abstract, spatial, temporal, physical, and planning reasoning
  • Programmatic Control: Systematic difficulty scaling through parameterized generation scripts
  • Embedded Questions: Questions integrated directly into video content, eliminating textual shortcuts
  • Evolving Benchmark: Controllable generation pipeline for creating new challenging instances as models improve

Reasoning Categories

Mathematical (16.8%)
Arithmetic operations, algebraic relations, quantitative comparisons
Abstract (12.8%)
Pattern recognition, logical inference, symbolic reasoning
Spatial (20.0%)
Object relationships, spatial transformations, 3D reasoning
Temporal (16.6%)
Sequence understanding, causal inference over time
Physical (14.0%)
Object dynamics, causal interactions, physics laws
Planning (19.8%)
Multi-step reasoning, goal-directed problem solving
